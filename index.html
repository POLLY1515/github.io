Buscador de CEP

O projeto consiste na criação de um buscador de CEP, utilizando a API Via CEP. Para a construção do designer,utilizei apenas Html e Css. Para o funcionamento da aplicação, utilizei javascript. O usuário,após colocar o Cep no campo específico,os campos logradouro e cidade ja serão preenchidos automaticamente,restando então ao usuário, preencher o campo número solicitado. Após todos os campos preenchidos,o usuário pode salvar o Cep cadastrado,ou limpar os campos.

  • Css
